# 🎿 Ski Resort Analysis & Recommender

## 📌 Overview

This project combines **Python**, **Power BI**, and **Streamlit** to analyze more than 5,000 ski resorts worldwide and help users find the destination that best matches their preferences.

The project includes data cleaning and exploratory analysis, an interactive Power BI dashboard, and a recommendation system capable of suggesting ski resorts based on budget, skier profile, and travel preferences.

---

## 🐍 Data Preparation & EDA

Using Python, the dataset was cleaned, transformed, and enriched through feature engineering. Missing values were handled, ski pass prices were standardized to EUR, and new metrics such as price per skiable kilometer were created.

Exploratory Data Analysis (EDA) was then performed to uncover trends and relationships between variables such as skiable terrain, snow reliability, altitude, pricing, and resort ratings.

---

## 📊 Power BI Dashboard

The Power BI dashboard provides an interactive overview of the global ski resort market, allowing users to explore destinations, compare resorts, and identify key trends through maps, KPIs, and advanced visualizations.

### Features

- Interactive filters
- KPI cards
- Geographic analysis
- Scatter plots
- Treemaps
- Resort comparisons
